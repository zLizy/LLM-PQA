Dataset Name: spotify_dataset

Overview:
The "spotify_dataset" contains data structured in several columns: 'user_id', 'artistname', 'trackname', and 'playlistname'. The dataset provides information on user interactions with various artists, tracks, and playlists on Spotify.

Potential Uses:
- Building recommendation models based on user interactions with artists and tracks.
