**Dataset Name**: NYC Property Sales

**Overview**: This dataset contains records of every building or building unit (apartment, etc.) sold in the New York City property market over a 12-month period from September 2016 to September 2017. It provides a detailed view of the NYC real estate market, including the location, address, type, sale price, and sale date of building units sold.

**Key Columns**:
- **BOROUGH**: A digit code for the borough the property is located in; in order these are Manhattan (1), Bronx (2), Brooklyn (3), Queens (4), and Staten Island (5).
- **NEIGHBORHOOD**: The neighborhood where the property is located.
- **BUILDING CLASS CATEGORY**: The type of building or building unit.
- **TAX CLASS AT PRESENT**: The current tax classification of the property.
- **BLOCK**: The block number of the property.
- **LOT**: The lot number of the property.
- **EASE-MENT**: Any easements on the property.
- **BUILDING CLASS AT PRESENT**: The current classification of the building.
- **ADDRESS**: The address of the property.

**Usage**: This dataset can be used for:
- Analyzing trends in the New York City real estate market.
- Building predictive models to forecast future property sales and prices.
- Identifying the most and least expensive neighborhoods in NYC.
- Evaluating the impact of building type and location on property prices.
